# sinon-in-sandbox

> Performs certain actions inside a Sinon sandbox

## Usage

```js
var sandbox = require('sinon-in-sandbox')

sandbox((sinon) => {
  // use sinon here. it'll automatically be cleaned up afterwards.
})
```
